# Binalbagan Catholic College (BCC) Landing Page

A modern, responsive one-page landing website for Binalbagan Catholic College featuring dynamic visual effects and interactive elements.

## Project Overview

This project showcases a complete landing page for BCC with multiple sections including Hero, Statistics, About, Academic Programs, Faculty, Departments, Department Details, and Contact. The website features modern design elements, animations, and interactive components.

## Technologies Used

- **HTML5** - Structure and semantic markup
- **CSS3** - Styling, animations, and visual effects
- **JavaScript** - Interactive features, animations, and form validation
- **Font Awesome 6.4.0** - Icons
- **Google Fonts (Poppins)** - Typography
- **Unsplash Images** - Hero background image and teacher avatars
